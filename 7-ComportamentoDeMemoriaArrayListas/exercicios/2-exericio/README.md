## Problema "soma_vetor"  
### Faça um programa que leia N números reais e armazene-os em um vetor. Em seguida: - Imprimir todos os elementos do vetor - Mostrar na tela a soma e a média dos elementos do vetor

### Quantos numeros voce vai digitar? __4__ 
### Digite um numero: __8.0__
### Digite um numero: __4.0__ 
### Digite um numero: __10.0__ 
### Digite um numero: __14.0__ 
-----
### VALORES = __8.0  4.0  10.0  14.0__   
### SOMA = __36.00__ 
### MEDIA = __9.00__
